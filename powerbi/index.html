Configuracion

  1. crear una cuenta que tenga asociada una licencia pro
  2. asignar en admin de office 365 los permisos de administrador de powerbi y power plataform
  3. ingresar al portal de azure con esa cuenta y registrar una nueva aplicación en registro de aplicaciones
  4. asignar permisos a las API de power bi
  5. ingresar al portal de powerbi y validar en Portal de Administración que este habilitado Conceder acceso a las APIs en Desarrollador
  6. Crear o modificar los workspace para darles acceso a la aplicación registrada en azure (verificar el guid)

Permisos

Se puede acceder para revisar desde el portal de azure -> Active Directory -> Registro de Aplicaciones -> Permisos de APIs El usuario con la cuenta pro debe tener establecido en roles de administrador:

  • Administrador de PowerBI
  • Administrador de Power Plataform
